Представляю завершенное тестовое задание.

1) Автотесты реализованы на Python3 и Selenium Webdriver. 
2) Использован тестовый фреймворк pytest. 
3) Реализован PageObject паттерн.
4) Включена генерация отчета с помощью pytest-html плагина.

В ходе работы использовался Chrome браузер. Драйвер предоставил в репозитории. 
Его необходимо добавить к рабочей папке.

Установлен pytest-html плагин для удобства генерации отчетов.
Инструкция по установке:
pip instal pytest-html
Тест запускается с параметром  --html=pytest_report.html
В последствии в рабочей папке создается файл с отчетом.
